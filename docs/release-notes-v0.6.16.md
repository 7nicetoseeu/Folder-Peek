# Folder Peek v0.6.16

这个版本新增了多种可选展开方式，并重构了应用内的 UI 页面。默认手势仍然是 `Space + 左键向任意方向拖动`。

## 新增

- 新增 `中键拖动`、`融合进右键菜单`、`长按左键`、`长按右键` 4 种展开方式，选择会自动保存。
- 使用说明顶部的手势图可以点击恢复默认手势；启用可选方式时，图片会暗化并模糊。
- 中键拖动支持首层与级联面板展开；修复快速拖放时最后一次移动丢失、导致无法级联的问题。
- 长按左键或右键达到 `500ms` 后会立即展开；级联面板仅支持短左键单击，避免长按误触。
- 右键菜单模式会注册“使用 Folder Peek 展开”，并支持已运行实例的路径转发。

## 改进

- 重构应用内 UI 页面，具体交互和页面预览请参考 README。

## 下载与使用

- 提供 `win-x64` framework-dependent 和 self-contained 两种发布包。
- framework-dependent 版本运行前需要已安装 **.NET 8 Desktop Runtime**。
- 运行后的设置、主题和索引缓存会写入 `%LocalAppData%\\FolderPeek`。
